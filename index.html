DAG Structure

Input -> f -> g -> h -> Output

Schedule

0 produce h:
1 parallel yo:
   for yo in [0, 5]
2   for xo in [0, 25]
3     for yi in [0, 25]
4       for xi in [0, 40]
5     produce g:
6     for g.yo in [0, 125]
7       for g.xo in [0, 991]
8         vectorized g.xi in [0,8]
9     vectorized h.xii in [0,8]

Algorithm

 Var x("x"), y("y");
 Func f("f"), g("g"), h("h");

 f(x, y) = (x + y) * (x + 2*y) * (x + 3*y) * (x + 4*y) * (x + 5*y);
 Expr e = 0;
 for (int i = 0; i < 100; i++) {
    e += f(x + i*10, y + i*10);
 }
 g(x, y) = e;
 e = 0;
 for (int i = 0; i < 100; i++) {
    e += g(x + i*10, y + i*10);
 }

 h(x, y) = e;
 h.estimate(x, 0, 1000).estimate(y, 0, 1000);
 Pipeline(h).auto_schedule(target, params);
